Regras do Jogo


        1- Separe as cartas de personagens das de suporte.
        2- Embaralhe -as  e deixe uma pilha para personagens e outra para suporte.
        3- Cada jogador vai pegar 5 cartas de personagens e 2 cartas de suporte.
        4- A batalha será 1x1, ou seja, 1 carta vs 1 carta.
        5- Os jogadores vão tirar Jokenpô para decidir quem vai jogar primeiro.
        6- O jogador que ganhar vai escolher uma de suas cartas normais para 
        iniciar. Será opcional anexar uma carta de suporte. 
        7- Depois que as duas cartas se atacarem os jogadores vão decidir se querem
        continuar com aquela carta no campo de batalha ou se vão trocar por outra do
        deck. 
        8- Quando uma carta "morre" o player que a matou ganha moedas. Com essas
        moedas é possível desbloquear uma habilidade especial nas cartas Pratas
        e  Ouro ou comprar uma nova carta.
                Se matar uma carta ouro:   3 moedas
                Se matar uma carta prata:  2 moedas
                Se matar uma carta bronze: 1 moeda

        8- Ganha o jogo quando alguém perder todas as cartas.


        
Observações
    
        - Contém 100 cartas (85 - personagens; 15 - Suporte )
        - Contém 50 moedas.
        - Até 2 jogadores. 

        1- Em toda a partida os jogadores podem ter no máximo 7 cartas (5 personagens e 2
        de suporte).
        2- As cartas de suporte podem ser anexadas a uma carta de personagem caso o jogador 
        queira. Elas deixam as cartas normais mais fortes.
        3- Ao anexar uma carta de suporte a um personagem ela ficará lá até o personagem morrer. 
        Depois disso ela é mandada para a pilha de descarte.
        4- Caso o player opte por comprar uma nova carta terá que descartar uma 
        de suas cartas na pilha de descarte, mas terá que descartar a carta com mais HP.
        5- O preço para comprar uma carta são 2 moedas.
        6- Para desbloquear uma habilidade 3 moedas.
